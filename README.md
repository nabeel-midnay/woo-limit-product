Plugin to set limited product numbers for WooCommerce.

## Description

Woo Limited Products allows store owners to limit the number of products a user can purchase and set expiration timers for these limits. This is ideal for flash sales, limited edition items, or managing inventory for high-demand products.

### Key Features
*   **Product Limits:** Set strict limits on the quantity of a product a user can purchase.
*   **Cart & Checkout Validation:** Prevents users from exceeding limits at both cart and checkout stages.
*   **Expiration Timers:** Set a time limit for how long a user can hold a "limited" product in their cart.
*   **Custom Order Status:** Includes support for custom order statuses like "Partially Shipped".
*   **Auto-cleanup:** Automatically cleans up expired blocks/reservations to free up inventory.

## Installation

1.  Upload the plugin files to the `/wp-content/plugins/woo-limit-product` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Configure the settings in the WooCommerce admin area.
